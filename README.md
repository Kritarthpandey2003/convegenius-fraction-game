# Interactive Fraction Game 🎮

Live Playable Game: https://convegenius-fraction-game.vercel.app/

An interactive, web-based educational game designed to help students learn fractions. Built as an assignment for the Product Intern role at ConveGenius.AI.

## Tech Stack
* **HTML5, CSS3, Vanilla JavaScript**: Clean, zero-dependency core logic.
* **Three.js**: Premium 3D visualization of the fraction models.
* **Web Audio & Web Speech APIs**: Procedurally generated sound effects and an integrated **AI Voice Assistant** for hands-free gameplay and text-to-speech instructions.

## Deliverables Included
* **Source Code**: Fully commented and organized in this repository.
* **Design Document**: Detailed in `Design_Document.md`.
* **Playable Game**: Open `index.html` to play instantly in any browser.

## How to Play
1. Look at the Target Fraction at the top of the screen.
2. Use the **+ PIECE** and **- PIECE** buttons to shade the 3D circular model.
3. Match the target fraction to trigger the success animation and progress to the next level!
